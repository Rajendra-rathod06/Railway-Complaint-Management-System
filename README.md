# Railway-Complaint-Management-System
A full-stack Railway Complaint Management System that allows passengers to submit complaints and enables administrators to track, manage, and resolve issues efficiently. The system supports image-based complaints, PNR tracking, and real-time status updates.



 🔥 Features
👤 User Module
Submit complaints with details
Upload complaint images 📷
Enter PNR number for tracking


🛠 Admin Module
View all complaints in dashboard
See uploaded images for verification
Update complaint status (Pending / In Progress / Resolved)


📊 Data Management
MySQL database integration
Stores complaint details securely
Efficient handling using file paths for images
